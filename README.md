# @ruforms/primitives

primitives for RuForms.

## Installation

```bash
pnpm add @ruforms/primitives
```

## Usage

```tsx
import { ExampleComponent } from '@ruforms/primitives'

function App() {
  return <ExampleComponent />
}
```

## Development

```bash
# Install dependencies
pnpm install

# Start development mode
pnpm dev

# Build for production
pnpm build

# Run linting
pnpm lint

# Type check
pnpm type-check
```
